# react-pokemon-api
reactでポケモンAPI

## 環境
- Windows 11 Pro 22H2
- node: 18.15.0

## プロジェクト作成、実行
```
# プロジェクト作成 
>  npx create-react-app react-pokemon-api

# プロジェクト実行
>  npm start
```
